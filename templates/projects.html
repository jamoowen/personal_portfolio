Park Boxing London

Website made for a personal training business, specializing in boxing training. Gets input from user, calls an API (find that postcode)to convert entered data (postal code) into GPS coordinates. Calls another API (geopy) to calculate the distance between the given GPS location and a constant to determine if client is within range of services. Additionally, implemented form using wtforms which takes input and sends an email to the business owners detailing the customer's queries and details.

Broking app

Web application made using flask. Has login and sign-up feature which stores user data in a relational database. Uses flask session to create a cookie on the user. Contacts IEX API to receive current share prices of a given stock, when the user searches for said stock. Allows user to buy the given stock and updates their cash and equity holdings within the database accordingly. Also allows user to sell stocks held and ensures the user has enough cash to fund purchases and enough stocks to sell. (Project completed as part of CS50 course and assistance was given)

DNA app

Application which initally takes two command line arguments to determine the given DNA sequence to be analyzed and a list of many different peoples' DNA sequences. App reads the csv files and then loops through the DNA sequences to determine which matches the sample. If the match is successfull, the name of the correct person is printed out and the function ends.

Password checker

Python app using the pwndedpasswords API. Takes command line arguments containing certain passwords. Hashes the given password through SHA1 and then runs the hashed password through a function which calls the API. The count of how many times the password has been leaked is returned and then finally printed to screen.
